Machete con funciones de tabla para Html

Funcion

Descripcion

Table

 Esta funcion es para generar la tabla y debe ir abierta y cerrada

Tr

 Agrega una fila a la tabla tambien debe ir abierta y cerrada

Td

 Para agregar datos a una fila

Th

 Para agregar el encabezado de una columna, dentro se agragan los datos

colspan, rowspan

 Es para expandir la columa o la fila a lo largo de la tabla

Thead, Tbody, Tfoot

 Es para definir en la tabla el encabezado, el cuerpo de la tabla y en pie de la tabla
